# Free Radio 自由电台 — NVDA 附加组件

FreeRadio 自由电台是一款专为 NVDA 屏幕阅读器设计的互联网电台附加组件。其主要目标是让用户能够轻松访问成千上万个互联网电台。整个界面和所有功能在设计时都充分考虑了 NVDA 的无障碍体验。

## 电台浏览器目录

自由电台使用 [Radio Browser](https://www.radio-browser.info/) 开源数据库作为其电台目录。Radio Browser 是一个由社区管理的免费目录，托管了来自全球 50,000 多个互联网电台。无需注册或账户，其 API 对所有人开放。每个电台都包含地址、国家、流派、语言和比特率信息；电台按用户投票进行排名。自由电台通过位于德国、荷兰和奥地利的镜像服务器连接到此 API；如果某台服务器无法访问，它会自动切换到下一台。

## 向 Radio Browser 添加电台

如果你查找的电台不在 Radio Browser 目录中，你可以自行在 [https://www.radio-browser.info/add](https://www.radio-browser.info/add) 提交。无需账户或注册。

填写该页面上的表格：

* **流媒体 URL (Stream URL)** *(必填)* — 音频流的直接 URL，以 `.mp3`, `.aac`, `.ogg` 或类似格式结尾。这不是电台的网站地址；它是你粘贴到媒体播放器中的原始流地址。大多数电台会在其网站或“在线收听”部分发布其流媒体 URL。


* **电台名称 (Station name)** *(必填)* — 电台应在目录中显示的名称。


* **主页 (Homepage)** — 电台的网站地址。


* **国家和语言 (Country and language)** — 从下拉列表中选择国家和广播语言。


* **标签 (Tags)** — 以逗号分隔的流派或主题关键词，例如 `news`（新闻）、`jazz`（爵士）、`classical`（古典）。这些用于搜索和过滤。


* **Logo URL** — 电台 Logo 图片的直接链接（如果可用）。



提交后，电台将经过审核并添加到公共目录中。一旦被接受，由于目录会从实时 API 刷新，它将自动出现在自由电台的搜索和国家/地区列表中。

## 电台浏览器

自由电台还在 NVDA 工具菜单中添加了一个 **自由电台** 子菜单。从那里你可以直接打开电台浏览器和自由电台设置。

使用 `Ctrl+Win+R` 打开的窗口包含五个选项卡：所有电台、收藏、录音、定时器和已点赞歌曲。你可以使用 `Ctrl+Tab` 在选项卡之间切换。

当“所有电台”选项卡打开时，来自 Radio Browser 的前 1,000 个最受欢迎电台会自动加载。从下拉列表中选择国家/地区将更新列表以显示该国家的电台。在搜索框中输入内容会立即同时按名称、国家和流派对整个 Radio Browser 数据库执行全面搜索。

浏览器窗口底部（选项卡外部）的 **输出设备 (Output Device)** 下拉列表列出了所有 BASS 识别的音频输出设备。选择设备会立即将音频输出重定向到该设备并永久保存选择；下次会话将自动使用同一设备。如果所选设备未连接，附加组件会自动回退到系统默认值。此控件仅在 BASS 后端处于活动状态时才起作用。

同一区域的 **音量 (Volume)**（0–200）和 **效果 (Effects)** 控件可以在窗口打开时的任何时间进行调整。从效果列表中，可以同时启用合唱 (Chorus)、压缩 (Compressor)、失真 (Distortion)、回声 (Echo)、镶边 (Flanger)、Gargle、混响 (Reverb)、EQ: 低音增强 (Bass Boost)、EQ: 高音增强 (Treble Boost) 和 EQ: 人声增强 (Vocal Boost)；更改将立即应用于活动音频流。这些控件仅在 BASS 后端处于活动状态时才能完全发挥作用。

当启用一个或多个 EQ 效果时，每个活动频段都会出现一个 **增益控制 (gain control)**。增益可在 −15 分贝 (dB) 到 +15 分贝 (dB) 之间设置；默认值为低音 +9 dB，高音 +9 dB，人声 +6 dB。增益控件仅针对当前选中的 EQ 频段显示，并在取消选中 EQ 效果时自动隐藏。增益值会全局保存并在下次会话中恢复。

**播放/暂停** 按钮也位于窗口底部。如果没有电台正在播放，它将启动所选电台；如果已有电台正在播放，则暂停播放。

在列表中选择电台时，**电台详细信息 (Station Details)** 按钮会在单独的对话框中显示国家、语言、流派、格式、比特率、网站和流媒体 URL 等信息。每个字段显示在各自的只读文本框中；你可以使用 Tab 在字段之间移动，并使用 **全部复制到剪贴板** 按钮一次性将所有信息复制到剪贴板。此按钮在“所有电台”和“收藏”选项卡中均可用。

## 收藏

收藏列表是一个永久存储的个人电台集合。要添加电台，请在列表中选择它并按“添加到收藏”按钮或使用 `Alt+V` 快捷键。再次使用该快捷键可移除已在列表中的电台。

收藏电台可以使用 `Ctrl+Win+→` 和 `Ctrl+Win+←` 进行播放；即使浏览器窗口未打开，这些快捷键也有效。

要从收藏列表中删除电台，请选中它并按“删除电台”按钮或 `Delete` 键。删除后，焦点和选择将自动移动到列表中的下一个电台。如果删除的是最后一个，焦点将移动到前一个。如果列表变为空，焦点将移动到“播放”按钮。

### 导出和导入收藏

“收藏”选项卡包含两个用于备份和恢复电台列表的按钮：

* **导出收藏...** — 将你的整个收藏列表保存到文件。保存对话框让你在两种格式之间选择：


* **JSON** (`.json`) — 完整的备份，保留了电台名称、流媒体 URL 和所有元数据。推荐用于稍后恢复列表或将其移动到另一台计算机。


* **M3U 播放列表** (`.m3u`) — 与大多数媒体播放器和电台应用程序兼容的标准播放列表格式。注意 M3U 不存储所有电台元数据，因此从 M3U 恢复可能比 JSON 备份包含的详细信息少。




* **导入收藏...** — 从先前导出的 JSON 或 M3U 文件加载电台。选择文件后，系统会询问你如何添加电台：


* **是 (合并)** — 将导入的电台添加到现有列表中，而不删除任何当前收藏。重复电台不会被重复添加。


* **否 (替换)** — 完全清除当前收藏列表并将其替换为导入文件的内容。


* **取消** — 返回浏览器而不做任何更改。





导入成功后，收藏列表、计划录音电台列表和定时器电台列表都会自动刷新。

### 重新排序收藏

在“收藏”选项卡中选中一个电台，按 `逗号` 键进入移动模式——你会听到一声提示音。使用箭头键导航到目标位置，然后再次按 `逗号` 键。电台将被放置在选定位置，新顺序会立即保存。在同一位置再次按 `逗号` 键可取消移动。

### 收藏电台的直接键盘快捷键

收藏列表中的每个电台都会在 NVDA 的“输入手势”对话框的 **自由电台电台** 类别下注册为单独的脚本。你可以为任何电台分配任何键盘快捷键，并从任何地方按下它——无需先打开浏览器窗口。

分配快捷键的方法：

1. 打开 NVDA 菜单 → 首选项 → 输入手势。


2. 展开 **自由电台电台** 类别。


3. 按名称找到电台，选中它，然后按“添加”。


4. 按下所需的键组合并确认。



快捷键会立即激活电台。如果以后从收藏中删除了该电台，其条目将从类别中消失，分配给它的任何快捷键也会被 NVDA 自动清除。当有新电台添加到收藏时，它会立即出现在该类别中——无需重新打开输入手势对话框。

### 添加自定义电台

要添加 Radio Browser 中不存在的电台，请使用“添加自定义电台”按钮。在出现的对话框中，输入电台名称和流媒体 URL，即可将其直接添加到收藏夹。自定义电台可以像其他收藏电台一样播放和重新排序。

### 电台音频配置文件

“收藏”选项卡包含两个用于管理每个电台音频设置的按钮：

* **为此电台保存音频配置文件** — 将当前音量级别、活动效果（合唱、EQ 等）和 EQ 增益值保存为绑定到该特定电台的配置文件。每当该电台开始播放时，其保存的音量、效果和增益设置会自动应用，覆盖全局默认值。


* **清除音频配置文件** — 从选定的电台移除已保存的音频配置文件。清除后，电台将恢复为全局音量、效果和 EQ 增益设置。仅当所选电台已有保存的配置文件时，此按钮才处于活动状态。



这两个按钮均位于收藏列表下方，仅在选中列表中的电台时启用。

## 音乐识别

按三次 `Ctrl+Win+I` 会触发基于 Shazam 的音乐识别功能，用于当前播放的音频流。识别仅在没有可用 ICY 元数据（电台广播的曲目信息）时启动；如果有元数据，则改为将其复制到剪贴板。

识别工作方式如下：使用 ffmpeg 从音频流中捕获简短的音频样本，应用 Shazam 指纹算法，并将结果发送到 Shazam 的服务器。如果识别成功，曲目名称、艺术家、专辑和发行年份将由 NVDA 播报并自动复制到剪贴板。如果启用了 **将已点赞歌曲保存到文本文件** 选项，识别结果还会附加到 `likedSongs.txt`。

**音频反馈：** 识别开始时会发出两声上升的提示音，结束时会发出两声下降的提示音。处理过程中，每 2 秒播放一次短促的提示音。

**要求：** 需要 `ffmpeg.exe`。放置在附加组件文件夹中的 `ffmpeg.exe` 会被自动使用；如果位于不同位置，则可以在设置中设置路径。从 [ffmpeg.org](https://ffmpeg.org/download.html) 下载 ffmpeg。

## 音频镜像

`Ctrl+Win+M` 快捷键可同时将当前播放的音频流镜像到第二个音频输出设备。这对于同时在两个不同的设备（如扬声器和耳机）上收听非常有用。

首次按键时，会出现一个列出可用输出设备的对话框。选择设备后，镜像开始，主播放不中断。再次按快捷键可停止镜像。

**使用场景：**

* **扬声器 + 耳机** — 让客人在耳机上跟随收听同样的广播，而你通过电脑扬声器收听。


* **录音设置** — 将主输出路由到扬声器，第二个输出路由到外部录音机或音频接口进行外部捕获。


* **多房间** — 同时通过蓝牙音箱和内置扬声器播放；无需额外的软件即可将音频传输到另一个房间。


* **远程监控** — 在屏幕共享或远程桌面会话中，本地和远程双方可以同时听到同样的音频流。



> **注意：** 音频镜像仅在 BASS 后端处于活动状态时才可用。如果在镜像活动时更改音量，两个输出将同时更新。

## 录音

录音默认保存到 `Documents\自由电台Recordings\`。文件名包含电台名称（或在歌曲录制模式下的曲目名称）以及录音开始时间。录音文件夹可以随时在“NVDA 菜单 → 首选项 → 设置 → 自由电台→ **录音文件夹**”中更改。由于录音引擎直接连接到音频流，音频在接收时即写入磁盘——不进行任何处理或重编码——录音质量与广播质量完全相同。

**即时录音：** 当电台播放时，按一次 `Ctrl+Win+E`。再次按键停止。播放始终不中断。

**歌曲录制：** 在播放广播 ICY 元数据的电台时，快速连续按**两次** `Ctrl+Win+E`。录制立即开始，并以当前曲目名称命名。当曲目更改时，录制自动停止，NVDA 播报保存的文件名。如果你想在曲目结束前提前结束录制，请再次按两次 `Ctrl+Win+E`。如果当前电台不广播 ICY 元数据，歌曲录制将不可用，NVDA 会通知你。

**计划录音：** 打开浏览器中的“录音”选项卡。从收藏中选择电台，以 HH:MM 格式输入开始时间，以分钟为单位输入持续时间，选择一个或多个活动日期，然后选择循环模式和录制模式：

* **活动日期：** 勾选一周中的一个或多个日期。在单次录制模式下，每个选定日期都会创建一个单独的条目，每个条目放置在下一次出现的该日期上。在循环模式下，录制仅在勾选的日期重复。如果不勾选任何日期，则录制不限于特定日期。


* **循环：**
* **单次录制** — 在每个选定日期录制一次。每个条目放置在下一次出现的该日期上；如果选定时间在今天已经过去，该条目会自动移至下周的同一天。


* **每周重复** — 在选定的活动日期每周重复，直到从计划列表中移除。




* **录制模式：**
* **边听边录** — 同时播放和录制。播放后端按 BASS → VLC → PotPlayer → Windows Media Player 的优先级顺序启动。


* **仅录制** — 在后台静默录制，无需任何音频输出；录音引擎直接连接到音频流。





NVDA 会在录音开始和结束时进行播报。如果 NVDA 在计划录音处于活动状态时重启，录音将在启动时自动恢复。

## 定时器

打开电台浏览器中的“定时器”选项卡 (`Alt+4`)。可以添加两种类型的定时器：

* **闹钟 — 启动电台：** 在指定时间自动开始播放从你的收藏中选定的电台。选择一个电台并以 HH:MM 格式输入时间。


* **睡眠 — 停止电台：** 在指定时间停止播放。当定时器触发时，音量会在播放停止前 60 秒内逐渐降低。无需选择电台；只需输入时间即可。



对于这两种类型，如果输入的时间已经过去，则该操作会被安排在第二天。如果已存在相同时间的定时器（任何类型），则阻止添加定时器；消息会通知你冲突并提示你先移除现有的条目。待处理的定时器列在选项卡中；选择一个并按“移除所选定时器”按钮即可取消它。

## 静音通知

当在设置中启用 **静音通知** 时，NVDA 会静音以下自动播报：

* 新电台开始播放时的名称


* 播放状态更改：播放、暂停、停止


* 录音事件：开始、停止、完成（即时、歌曲和计划录音）


* ICY 曲目更改播报，即使 **自动播报曲目更改** 也已启用



以下播报不受影响：错误消息、收藏反馈（已添加/已在列表中）、音乐识别结果和更新通知。

该设置可以通过“NVDA 菜单 → 首选项 → 设置 → FreeRadio”进行切换，或通过未分配的输入手势随时即时切换。切换时，NVDA 会播报“通知已静音”或“通知已取消静音”以确认更改。

## 自动播报曲目更改

当在设置中启用 **自动播报曲目更改** 选项时，自由电台大约每 5 秒在后台检查活动电台的 ICY 元数据流。当曲目更改时，新标题由 NVDA 自动读取——无需按键。

切换到新电台时，一旦建立连接，就会立即播报第一条曲目信息。如果你切换到不广播 ICY 元数据的电台，系统保持静默，且不会重复之前电台的曲目信息。

此功能默认禁用，可以通过“NVDA 菜单 → 首选项 → 设置 → FreeRadio”进行切换。

## 已点赞歌曲

当启用 **将已点赞歌曲保存到文本文件** 选项时，通过按三次 `Ctrl+Win+I` 复制到剪贴板的曲目信息也会逐行附加到 `Documents\自由电台Recordings\likedSongs.txt`。

在广播 ICY 元数据的电台上，曲目标题和艺术家直接保存。在没有 ICY 元数据的电台上，Shazam 识别结果保存到同一个文件——两个来源共享同一个列表。如果文件不存在，会自动创建；每个条目都会附加到文件末尾，之前的条目永远不会被删除。

## 已点赞歌曲选项卡

电台浏览器中的 **已点赞歌曲** 选项卡显示 `likedSongs.txt` 中保存的所有曲目。每次打开该选项卡时，列表都会自动从文件重新加载。

列表上方的 **过滤 (Filter)** 字段让你实时缩小显示的曲目范围。输入曲目标题或艺术家名称的任何部分，列表会在每次按键时立即更新。NVDA 在每次更改后播报匹配结果的数量。从过滤字段按 `向下` 箭头可将焦点直接移至列表中。

从列表中选择曲目可启用以下操作：

* **在 Spotify 上播放：** 尝试直接打开 Spotify 桌面应用。如果未安装该应用，则回退到 Spotify 网站并自动开始播放第一个结果。


* **在 YouTube 上播放 (`Alt+O`)：** 在默认浏览器中搜索所选曲目并打开结果。


* **显示歌词：** 获取并显示所选曲目的歌词。歌词从 [lrclib.net](https://lrclib.net) 获取（免费，无需账户）。当后台搜索运行时，会播报简短的“正在获取歌词...”消息。如果找到歌词，它们会在只读对话框中打开，你可以使用 NVDA 阅读并复制到剪贴板。如果没有找到歌词，NVDA 会通知你。此按钮在获取过程中暂时禁用，以防止重复请求。


* **移除 (`Alt+M`)：** 从 `likedSongs.txt` 中删除选定的曲目并更新列表。列表聚焦时，`Delete` 键也会触发此按钮。


* **刷新 (`Alt+E`)：** 从文件重新加载列表。



Spotify、YouTube、显示歌词和移除按钮仅在列表中选定实际曲目时启用。

### 歌词服务

自由电台使用 [lrclib.net](https://lrclib.net) 来获取歌词——这是一个免费、开放的数据库，不需要 API 密钥或账户。查找过程解析存储在 `likedSongs.txt` 中的曲目字符串，并尝试逐渐放宽查询，直到找到歌词：

1. 使用完整艺术家名称和清理后的标题精确匹配（在搜索前剔除噪声后缀，如“Remastered”、“Live”或年份标签）。


2. 使用完整艺术家名称和原始标题精确匹配（如果清理更改了它）。


3. 仅使用第一个艺术家名称和清理后的标题精确匹配（用于多艺术家字符串，如“Artist A & Artist B”）。


4. 使用第一个艺术家名称和清理后的标题进行模糊搜索。


5. 作为最后手段，使用原始曲目字符串进行模糊搜索。



当普通歌词可用时，它们按原样显示。当仅有时间同步的 LRC 歌词可用时，时间戳被剔除并显示纯文本。纯音乐曲目被报告为未找到。

## 播放

附加组件按以下优先级顺序选择播放后端：

1. **BASS** — 默认且主要的后端。无需单独安装；它与附加组件捆绑在一起。BASS 将音频直接发送到 Windows 音频堆栈，并在 Windows 音量混合器中显示为一个独立的音频源，名为“pythonw.exe”，与 NVDA 分开。这意味着自由电台音频在与 NVDA 语音完全分开的通道上流动：收音机不会在 NVDA 说话时被中断、混合或受到 NVDA 自身音频设置的影响。用户可以在 Windows 音量混合器中独立于 NVDA 调节收音机音量。支持 HTTP、HTTPS 和大多数嵌入式流格式。音频镜像仅在此后端可用。


2. **VLC** — 如果 BASS 失败则接管。在常用安装位置、用户配置文件文件夹和系统 PATH 中自动搜索。


3. **PotPlayer** — 如果未找到 VLC 则尝试。在常用安装位置自动搜索。


4. **Windows Media Player** — 作为最后手段使用；要求系统上安装 WMP 组件。



## 更新检查

自由电台通过 GitHub 自动检查新版本。

* **自动检查：** NVDA 启动 15 秒后在后台静默运行。如果找到新版本，你会收到通知；如果没有找到，则不显示消息。


* **手动检查：** 可以从“NVDA 工具 → 自由电台→ **检查更新...**”按需触发。以这种方式启动时，即使版本是最新的，也会播报结果。


* **找到更新时：** 会打开一个显示版本号和已安装版本的对话框。


* 如果 GitHub 发行版上有可直接下载的 `.nvda-addon` 文件，则显示“下载并安装”按钮。确认后，文件在后台下载，NVDA 会在下载开始时播报，并且 NVDA 自身的安装屏幕会自动打开。


* 如果没有直接下载链接，则显示“打开页面”按钮，GitHub 发行页面会在默认浏览器中打开。




* **禁用自动检查：** 从“NVDA 菜单 → 首选项 → 设置 → FreeRadio”关闭 **自动检查更新** 选项。

## 时移（直播收音机后退）

时移允许你后退当前收听的电台，就像DVR或磁带——暂停当下，回退几分钟，随时追上直播。播放无需停止：后退和快进在同一音频流上即时进行。

此功能**默认禁用**。在"NVDA 菜单 → 首选项 → 设置 → 自由电台 → **启用时移缓冲区（直播收音机后退，约10分钟）**"中启用，或随时使用`Ctrl+Win+T`即时切换。

### 工作原理

启用后，FreeRadio在后台将当前播放的电台持续捕获到滚动的本地缓冲区中。缓冲区保存大约**最近10分钟**的音频；新音频到来时旧音频从头部自动丢弃，因此缓冲区始终代表相对于直播边缘的"最近过去"。

- **`Ctrl+Win+J`** — 后退15秒。第一次按键将你从直播播放切换到时移播放，从直播边缘后退15秒开始。每次额外按键再后退15秒。
- **`Ctrl+Win+K`** — 在时移模式下快进15秒。到达直播边缘后，播放自动切换回直播流，NVDA播报"返回直播"。
- **`Ctrl+Win+T`** — 开启或关闭整个功能。在时移模式下关闭会立即返回直播并停止当前电台的后台捕获。

后台捕获在整个时移期间持续运行，所以即使你在收听几分钟前的内容，直播边缘仍在向前推进——和真正的DVR完全一样。

### 启用与缓冲区预热

缓冲区在电台开始播放时（功能启用后）或你在收听电台时启用功能的那一刻开始填充。因此，只有在实际捕获了几秒音频后才能后退——如果在切换电台后立即按`Ctrl+Win+J`，NVDA会告诉你缓冲区中还没有足够的音频。等待几秒再试即可。

切换到不同电台总会为新电台重置缓冲区；前一电台的缓冲音频被丢弃。

### 支持的流

时移适用于FreeRadio已支持的同类流：

- 普通HTTP/HTTPS流（MP3、AAC、OGG等），包括Shoutcast/Icecast风格的服务器。
- **HLS（`.m3u8`）流** — FreeRadio解析电台的主播放列表，跟踪媒体播放列表，并在后台下载片段以保持缓冲区填充。

在电台播放列表完全无法读取的罕见情况下（例如损坏或无法访问的`.m3u8`清单），NVDA会告诉你该特定电台不支持后退。

### 要求和限制

- **需要BASS后端。** 禁用BASS时时移不可用。
- 缓冲区约为10分钟；不能后退超过此限制。
- 缓冲区按电台区分：切换电台、停止播放或重启NVDA会清除缓冲区并重新开始。
