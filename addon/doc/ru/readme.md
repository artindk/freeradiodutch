# FreeRadio (Свободное радио)

FreeRadio — это интернет-радио-дополнение для экранного диктора NVDA. Его основная цель — предоставить пользователям простой доступ к тысячам интернет-радиостанций. Весь интерфейс и все функции разработаны с учётом полной доступности для NVDA.

## Каталог Radio Browser

FreeRadio использует открытую базу данных [Radio Browser](https://www.radio-browser.info/) для своего каталога станций. Radio Browser — это общедоступный бесплатный каталог, содержащий более 50 000 интернет-радиостанций со всего мира. Регистрация или учётная запись не требуются, а его API открыт для всех. Каждая станция включает адрес, страну, жанр, язык и информацию о битрейте; станции ранжируются по голосам пользователей. FreeRadio подключается к этому API через зеркальные серверы, расположенные в Германии, Нидерландах и Австрии; если один сервер недоступен, дополнение автоматически переключается на следующий.

## Добавление станции в Radio Browser

Если станция, которую вы ищете, отсутствует в каталоге Radio Browser, вы можете добавить её самостоятельно по адресу [https://www.radio-browser.info/add](https://www.radio-browser.info/add). Учётная запись или регистрация не требуются.

Заполните форму на этой странице:

- **Stream URL** *(обязательно)* — прямой URL-адрес аудиопотока, заканчивающийся на `.mp3`, `.aac`, `.ogg` или подобное. Это не адрес сайта станции; это «сырой» адрес потока, который вы вставили бы в медиаплеер. Большинство станций публикуют свой URL-адрес потока на своём веб-сайте или в разделе «Слушать онлайн».
- **Station name** *(обязательно)* — название станции, как оно должно отображаться в каталоге.
- **Homepage** — адрес веб-сайта станции.
- **Country and language** — выберите страну и язык вещания из раскрывающихся списков.
- **Tags** — ключевые слова жанра или темы, разделённые запятыми, например `news`, `jazz`, `classical`. Они используются для поиска и фильтрации.
- **Logo URL** — прямая ссылка на логотип станции, если доступен.

После отправки станция проверяется и добавляется в общедоступный каталог. После принятия она автоматически появится в результатах поиска и списках стран FreeRadio, поскольку каталог обновляется из живого API.

## Требования

- NVDA 2024.1 или новее
- Windows 10 или новее
- Интернет-соединение

## Установка

Скачайте `.nvda-addon` файл, нажмите на нём Enter и перезапустите NVDA при появлении запроса.

## Горячие клавиши

Все сочетания клавиш можно переназначить в меню NVDA → Параметры → Жесты ввода → FreeRadio. Эти сочетания работают отовсюду, независимо от того, какое окно находится в фокусе.

| Сочетание | Функция | Описание |
|---|---|---|
| `Ctrl+Win+R` | Открыть обозреватель станций | Открывает окно обозревателя, если оно закрыто, или выводит его на передний план, если уже открыто. |
| `Ctrl+Win+P` | Пауза / возобновление | Ставит текущую станцию на паузу, если играет; возобновляет, если на паузе. Если ничего не играет, запускает последнюю станцию или открывает список избранного в зависимости от вашей настройки. Двойное нажатие переключает на вкладку по вашему выбору. Тройное нажатие может вызвать отдельное действие в зависимости от вашей настройки. |
| `Ctrl+Win+S` | Остановить | Полностью останавливает текущую станцию и сбрасывает плеер. |
| `Ctrl+Win+→` | Следующая в избранном | Переходит к следующей станции в списке избранного. В конце списка переходит к началу. |
| `Ctrl+Win+←` | Предыдущая в избранном | Переходит к предыдущей станции в списке избранного. В начале списка переходит в конец. |
| `Ctrl+Win+↑` | Увеличить громкость | Увеличивает громкость на 10; максимум 100. |
| `Ctrl+Win+↓` | Уменьшить громкость | Уменьшает громкость на 10; минимум 0. |
| `Ctrl+Win+V` | Добавить в избранное | Добавляет текущую играющую станцию в список избранного. Сообщает, если станция уже в списке. |
| `Ctrl+Win+I` | Информация о станции | Объявляет название текущей играющей станции. Нажмите дважды, чтобы показать подробности, такие как страна, жанр и битрейт, в диалоговом окне. | `Ctrl+Win+M` | Аудиозеркало | Зеркально отражает текущий поток на дополнительное устройство вывода звука одновременно. Нажмите снова, чтобы остановить зеркалирование. |
| `Ctrl+Win+E` | Мгновенная запись | Нажмите один раз, чтобы начать запись текущей станции; нажмите снова, чтобы остановить. Нажмите **дважды**, чтобы начать **запись песни** — файл будет назван по текущему треку, и запись автоматически остановится при смене трека. Нажмите дважды снова, пока активна запись песни, чтобы остановить её досрочно. Воспроизведение продолжается без прерываний во всех режимах записи. Доступно только для станций, которые передают ICY-метаданные. |
| `Ctrl+Win+W` | Открыть папку записей | Открывает папку с сохранёнными файлами записей в Проводнике. |
| *(не назначено)* | Переключить отключение уведомлений | Переключает параметр «Отключить уведомления» на лету. Назначьте сочетание клавиш через меню NVDA → Параметры → Жесты ввода → FreeRadio. |
| *(не назначено)* | Воспроизвести избранную станцию напрямую | Каждая станция из списка избранного отображается как отдельный элемент в меню NVDA → Параметры → Жесты ввода → **FreeRadio Stations**. Назначьте сочетание клавиш любой станции и запускайте её мгновенно из любого места, не открывая обозреватель. |

Сочетания «следующий / предыдущий» работают только со списком избранного; они не работают со списком всех станций. Когда в окне обозревателя фокус находится на списке, клавиши со стрелками влево и вправо служат той же цели — см. «Горячие клавиши внутри диалога».

## Обозреватель станций

FreeRadio также добавляет подменю **FreeRadio** в меню NVDA «Сервис». Оттуда вы можете напрямую открыть Обозреватель станций и Настройки FreeRadio.

Окно, открываемое с помощью `Ctrl+Win+R`, содержит пять вкладок: Все станции, Избранное, Запись, Таймер и Понравившиеся треки. Вы можете перемещаться между вкладками с помощью `Ctrl+Tab`.

Когда открывается вкладка «Все станции», из Radio Browser автоматически загружаются 1000 самых популярных станций (по голосам). Выбор страны из раскрывающегося списка обновляет список, показывая станции этой страны. Ввод текста в поле поиска мгновенно выполняет полноценный поиск по всей базе данных Radio Browser одновременно по названию, стране и жанру.

Раскрывающийся список **Устройство вывода** в нижней части окна обозревателя (за пределами вкладок) содержит все распознаваемые BASS устройства вывода звука. Выбор устройства немедленно перенаправляет на него аудиовыход и сохраняет выбор навсегда; то же устройство будет использоваться автоматически в следующем сеансе. Если выбранное устройство не подключено, дополнение автоматически переключается на системное устройство по умолчанию. Этот элемент управления работает только когда активен движок BASS.

Элементы управления **Громкость** (0–200) и **Эффекты** в той же области можно регулировать в любое время, пока окно открыто. Из списка Эффектов можно одновременно включить Chorus, Compressor, Distortion, Echo, Flanger, Gargle, Reverb, EQ: Bass Boost, EQ: Treble Boost и EQ: Vocal Boost; изменения применяются к активному потоку мгновенно. Эти элементы управления полностью функциональны только когда активен движок BASS.

При активации одного или нескольких эффектов эквалайзера автоматически появляется **элемент управления усилением** для каждого активного диапазона. Усиление можно настроить в диапазоне от −15 дБ до +15 дБ; значения по умолчанию: НЧ +9 дБ, ВЧ +9 дБ, вокал +6 дБ. Элементы управления отображаются только для отмеченных диапазонов EQ и автоматически скрываются при снятии отметки с эффекта. Значения сохраняются постоянно и восстанавливаются в следующем сеансе.

Кнопка **Воспроизведение/Пауза** также находится в нижней части окна. Если ни одна станция не играет, она запускает выбранную станцию; если станция уже играет, она ставит воспроизведение на паузу.

Когда станция выбрана в списке, кнопка **Сведения о станции** отображает такую информацию, как страна, язык, жанр, формат, битрейт, веб-сайт и URL-адрес потока в отдельном диалоговом окне. Каждое поле отображается в отдельном текстовом поле только для чтения; вы можете перемещаться между полями с помощью Tab и скопировать всю информацию в буфер обмена сразу с помощью кнопки **Копировать всё в буфер обмена**. Эта кнопка доступна как на вкладке «Все станции», так и на вкладке «Избранное».

## Избранное

Список избранного — это персональная коллекция станций, которая хранится постоянно. Чтобы добавить станцию, выберите её в списке и нажмите кнопку «Добавить в избранное» или используйте сочетание `Alt+V`. То же сочетание удаляет станцию, которая уже есть в списке, когда она выбрана.

Избранными станциями можно управлять с помощью `Ctrl+Win+→` и `Ctrl+Win+←`; эти сочетания работают, даже когда окно обозревателя закрыто.

Чтобы удалить станцию из списка избранного, выберите её и нажмите кнопку **Удалить станцию** или клавишу `Delete`. После удаления фокус и выделение автоматически перемещаются на следующую станцию в списке. Если удалённая станция была последней, фокус перемещается на предыдущую станцию. Если список становится пустым, фокус перемещается на кнопку «Воспроизвести».

### Экспорт и импорт избранного

Вкладка «Избранное» содержит две кнопки для резервного копирования и восстановления списка станций:

**Экспортировать избранное…** — сохраняет весь список избранного в файл. В диалоге сохранения можно выбрать один из двух форматов:
- **JSON** (`.json`) — полная резервная копия, сохраняющая названия станций, URL-адреса потоков и все метаданные. Рекомендуется для последующего восстановления списка или переноса его на другой компьютер.
- **Плейлист M3U** (`.m3u`) — стандартный формат плейлиста, совместимый с большинством медиаплееров и радиоприложений. Обратите внимание, что M3U не хранит все метаданные станций, поэтому восстановление из M3U может дать меньше деталей, чем резервная копия JSON.

**Импортировать избранное…** — загружает станции из ранее экспортированного файла JSON или M3U. После выбора файла предлагается выбрать способ добавления станций:
- **Да (Объединить)** — добавляет импортируемые станции к существующему списку, не удаляя текущие элементы избранного. Дубликаты не добавляются повторно.
- **Нет (Заменить)** — полностью очищает текущий список избранного и заменяет его содержимым импортируемого файла.
- **Отмена** — возвращает в браузер без внесения каких-либо изменений.

После успешного импорта список избранного, список станций для запланированных записей и список станций таймера обновляются автоматически.

### Изменение порядка в избранном

Когда станция выбрана на вкладке «Избранное», нажмите `запятую`, чтобы войти в режим перемещения — вы услышите звуковой сигнал. Переместитесь на нужную позицию с помощью клавиш со стрелками, затем снова нажмите `запятую`. Станция будет помещена на выбранную позицию, и новый порядок будет немедленно сохранён. Повторное нажатие `запятой` на той же позиции отменяет перемещение.

### Прямые горячие клавиши для станций из избранного

Каждая станция из списка избранного зарегистрирована как отдельный скрипт в диалоге «Жесты ввода» NVDA, в категории **FreeRadio Stations**. Вы можете назначить любое сочетание клавиш любой станции и нажимать его из любого места — без необходимости открывать окно обозревателя.

Чтобы назначить сочетание:

1. Откройте меню NVDA → Параметры → Жесты ввода.
2. Разверните категорию **FreeRadio Stations**.
3. Найдите станцию по названию, выберите её и нажмите **Добавить**.
4. Нажмите нужное сочетание клавиш и подтвердите.

Сочетание запускает станцию немедленно. Если станция будет удалена из избранного, её запись исчезнет из категории, а назначенное сочетание будет автоматически удалено NVDA. При добавлении новой станции в избранное она сразу появляется в категории — повторно открывать диалог «Жесты ввода» не нужно.

### Добавление пользовательской станции

Чтобы добавить станцию, которой нет в Radio Browser, используйте кнопку «Добавить пользовательскую станцию». В появившемся диалоговом окне введите название станции и URL-адрес потока, чтобы добавить её прямо в избранное. Пользовательские станции можно воспроизводить и изменять их порядок так же, как и любую другую станцию из избранного.

### Аудиопрофиль станции

Вкладка «Избранное» содержит две кнопки для управления настройками звука для каждой станции:

**Сохранить аудиопрофиль для этой станции** — сохраняет текущий уровень громкости, активные эффекты и значения усиления эквалайзера как профиль, привязанный к этой конкретной станции. Всякий раз, когда эта станция начинает играть, её сохранённые громкость, эффекты и усиление автоматически применяются, переопределяя глобальные настройки по умолчанию.

**Очистить аудиопрофиль** — удаляет сохранённый аудиопрофиль для выбранной станции. После очистки станция возвращается к глобальным настройкам громкости, эффектов и усиления эквалайзера. Эта кнопка активна, только если для выбранной станции уже сохранён профиль.

Обе кнопки расположены под списком избранного и активны, только когда станция в списке выбрана.

## Распознавание музыки

Тройное нажатие `Ctrl+Win+I` запускает распознавание музыки на основе Shazam для текущего потока. Распознавание запускается только тогда, когда нет доступных ICY-метаданных (информация о треке, передаваемая станцией); если метаданные присутствуют, они копируются в буфер обмена.

Распознавание работает следующим образом: из потока захватывается короткий аудиообразец с помощью ffmpeg, применяется алгоритм создания отпечатков Shazam, и результат отправляется на серверы Shazam. Если распознавание успешно, название трека, исполнитель, альбом и год выпуска объявляются NVDA и автоматически копируются в буфер обмена. Если включена опция **Сохранять понравившиеся треки в текстовый файл**, результат распознавания также добавляется в `likedSongs.txt`.

**Аудиообратная связь:** Два повышающихся звуковых сигнала означают начало распознавания, и два понижающихся — его окончание. Короткий звуковой сигнал воспроизводится каждые 2 секунды во время выполнения процесса.

**Требование:** Требуется ffmpeg.exe. FFmpeg.exe, помещённый в папку дополнения, используется автоматически; если он находится в другом месте, путь можно указать в настройках. Скачайте ffmpeg с [ffmpeg.org](https://ffmpeg.org/download.html).

## Аудиозеркало

Сочетание `Ctrl+Win+M` зеркально отражает текущий поток на второе устройство вывода звука одновременно. Это полезно для одновременного прослушивания на двух разных устройствах, например, на колонках и наушниках.

При первом нажатии появляется диалог выбора с перечислением доступных устройств вывода. Как только устройство выбрано, начинается зеркалирование, а основное воспроизведение продолжается без прерываний. Повторное нажатие сочетания останавливает зеркалирование.

**Варианты использования:**
- **Колонки + наушники** — Гость может слушать ту же трансляцию в наушниках, пока вы слушаете через колонки компьютера.
- **Настройка записи** — Направьте основной выход на колонки, а второй выход на внешний рекордер или аудиоинтерфейс для внешней записи.
- **Мультирум** — Воспроизводите через Bluetooth-колонку и встроенные динамики одновременно; не требуется дополнительного программного обеспечения для передачи звука в другую комнату.
- **Удалённый мониторинг** — В сеансе демонстрации экрана или удалённого рабочего стола и локальная, и удалённая стороны могут слышать один и тот же поток одновременно.

> **Примечание:** Аудиозеркало доступно только когда активен движок BASS. Если громкость изменяется во время активного зеркалирования, оба выхода обновляются одновременно.

## Запись

Записи сохраняются в `Documents\FreeRadio Recordings\` по умолчанию. Имя файла включает название станции (или название песни в режиме записи песни) и время начала записи. Папку записей можно изменить в любое время через меню NVDA → Параметры → Настройки → FreeRadio → **Папка записей**. Поскольку механизм записи подключается непосредственно к потоку, аудио записывается на диск по мере получения — без обработки или перекодирования; качество записи идентично качеству трансляции.

**Мгновенная запись:** Пока станция играет, нажмите `Ctrl+Win+E` один раз. Нажмите снова, чтобы остановить. Воспроизведение продолжается без прерываний на протяжении всего процесса.

**Запись песни:** Нажмите `Ctrl+Win+E` **дважды** подряд, пока играет станция, которая передаёт ICY-метаданные. Запись начинается немедленно и называется по текущему названию трека. Когда трек меняется, запись автоматически останавливается, и NVDA объявляет сохранённое имя файла. Если вы хотите закончить запись досрочно, до окончания трека, снова нажмите `Ctrl+Win+E` дважды. Если текущая станция не передаёт ICY-метаданные, запись песни недоступна, и NVDA сообщит вам об этом.

**Запланированная запись:** Откройте вкладку «Запись» в обозревателе станций. Выберите станцию из вашего избранного, введите время начала в формате ЧЧ:ММ и продолжительность в минутах, выберите один или несколько активных дней, затем задайте режим повтора и режим записи:

**Активные дни:** Отметьте один или несколько дней недели. В режиме однократной записи для каждого выбранного дня создаётся отдельная запись, установленная на ближайшее наступление этого дня. В режиме повтора запись выполняется только в отмеченные дни. Если дни не выбраны, запись не ограничивается конкретными днями.

**Режим повтора:**
- **Записать один раз** — создаёт однократную запись для каждого выбранного дня. Каждая запись устанавливается на ближайшее наступление этого дня; если сегодняшнее время уже прошло, запись автоматически переносится на следующую неделю.
- **Повторять еженедельно** — повторяется каждую неделю в выбранные активные дни до удаления из списка расписания.

**Режим записи:**
- **Записывать во время прослушивания** — воспроизводит и записывает одновременно. Запускается движок воспроизведения в приоритетном порядке BASS → VLC → PotPlayer → Windows Media Player.
- **Только запись** — записывает в фоновом режиме без вывода звука; механизм записи подключается непосредственно к потоку.

NVDA объявляет, когда запись начинается и когда заканчивается. Если NVDA перезапускается во время активной запланированной записи, запись автоматически возобновляется при запуске.

## Таймер

Откройте вкладку «Таймер» в обозревателе станций (`Alt+4`). Можно добавить два типа таймеров:

**Будильник — включить радио:** Автоматически начинает воспроизводить выбранную станцию из избранного в указанное время. Выберите станцию и введите время в формате ЧЧ:ММ.

**Сон — выключить радио:** Останавливает воспроизведение в указанное время. Когда таймер срабатывает, громкость постепенно уменьшается в течение 60 секунд перед остановкой воспроизведения. Выбирать станцию не нужно; просто введите время.

Для обоих типов, если введённое время уже прошло, действие планируется на следующий день. Если в то же время уже существует таймер (независимо от типа), добавление нового заблокировано; пользователь получает уведомление о конфликте и предложение сначала удалить существующую запись. Ожидающие таймеры перечислены на вкладке; выберите один и нажмите кнопку «Удалить выбранный таймер», чтобы отменить его.

Список включает все совместимые с BASS устройства в системе, а также опцию «Системное устройство по умолчанию». Изменения применяются немедленно при сохранении; если выбранное устройство отключено, дополнение автоматически переключается на системное устройство по умолчанию и объявляет об изменении. ## Отключение уведомлений

Когда в Настройках включен параметр **Отключить уведомления**, NVDA не озвучивает следующие автоматические объявления:

- Название станции при начале воспроизведения новой станции
- Изменения состояния воспроизведения: воспроизведение, пауза, остановка
- События записи: начата, остановлена, завершена (мгновенная, запись песни и запланированная)
- Объявления о смене ICY-треков, даже если также включена опция **Автоматически объявлять смену треков**

Следующие объявления намеренно **не затрагиваются**: сообщения об ошибках, обратная связь об избранном (добавлена / уже в списке), результаты распознавания музыки и уведомления об обновлениях.

Параметр можно переключить в меню NVDA → Параметры → Настройки → FreeRadio или мгновенно в любое время через неназначенный жест ввода (назначьте его в меню NVDA → Параметры → Жесты ввода → FreeRadio). При переключении NVDA один раз объявляет «Уведомления отключены» или «Уведомления включены» для подтверждения изменения.

## Автоматическое объявление смены треков

Когда в Настройках включена опция **Автоматически объявлять смену треков**, FreeRadio проверяет поток ICY-метаданных активной станции в фоновом режиме примерно каждые 5 секунд. Когда трек меняется, новое название автоматически озвучивается NVDA — нажатие клавиш не требуется.

При переключении на новую станцию первая информация о треке объявляется сразу после установления соединения. Если вы переключитесь на станцию, которая не передаёт ICY-метаданные, система молчит, и информация о треке с предыдущей станции не повторяется.

Эта функция отключена по умолчанию и может быть включена в меню NVDA → Параметры → Настройки → FreeRadio.

## Понравившиеся треки

Когда включена опция **Сохранять понравившиеся треки в текстовый файл**, информация о треке, скопированная в буфер обмена троекратным нажатием `Ctrl+Win+I`, также добавляется построчно в `Documents\FreeRadio Recordings\likedSongs.txt`.

На станциях, передающих ICY-метаданные, название трека и исполнитель сохраняются напрямую. На станциях без ICY-метаданных результат распознавания Shazam сохраняется в тот же файл — оба источника используют один общий список. Файл создаётся автоматически, если его не существует; каждая запись добавляется в конец файла, предыдущие записи никогда не удаляются.

## Вкладка «Понравившиеся треки»

Вкладка **«Понравившиеся треки»** в браузере станций отображает все треки, сохранённые в `likedSongs.txt`. Список автоматически перезагружается из файла каждый раз при открытии вкладки.

Поле **Фильтр** над списком позволяет сужать отображаемые треки в реальном времени. Введите любую часть названия песни или имени исполнителя — список мгновенно обновляется после каждого нажатия клавиши. NVDA объявляет количество совпадающих результатов после каждого изменения. Нажмите стрелку `Вниз` в поле фильтра, чтобы переместить фокус непосредственно в список.

При выборе трека из списка становятся доступны следующие действия:

- **Воспроизвести в Spotify:** Пытается напрямую открыть настольное приложение Spotify. Если приложение не установлено, переходит на сайт Spotify и автоматически начинает воспроизводить первый результат.
- **Воспроизвести на YouTube (`Alt+O`):** Выполняет поиск выбранного трека на YouTube и открывает результаты в браузере по умолчанию.
- **Показать текст песни:** Загружает и отображает текст выбранного трека. Текст песни загружается с [lrclib.net](https://lrclib.net) (бесплатно, без регистрации). Пока поиск выполняется в фоновом режиме, объявляется короткое сообщение «Загрузка текста…». Если текст найден, он открывается в диалоговом окне только для чтения, где его можно читать с помощью NVDA и копировать в буфер обмена. Если текст не найден, NVDA сообщает об этом. Кнопка временно отключается во время выполнения загрузки, чтобы предотвратить дублирующие запросы.
- **Удалить (`Alt+M`):** Удаляет выбранный трек из `likedSongs.txt` и обновляет список. Клавиша `Delete` также активирует эту кнопку, когда список находится в фокусе.
- **Обновить (`Alt+E`):** Перезагружает список из файла.

Кнопки Spotify, YouTube, «Показать текст песни» и «Удалить» активны только тогда, когда в списке выбран реальный трек.

### Служба текстов песен

FreeRadio использует [lrclib.net](https://lrclib.net) для получения текстов песен — бесплатная открытая база данных, не требующая ключа API или учётной записи. Процесс поиска анализирует строку трека, хранящуюся в `likedSongs.txt`, и последовательно пробует более широкие запросы до нахождения текста:

1. Точное совпадение с полным именем исполнителя и очищенным названием (шумовые суффиксы, такие как «Remastered», «Live» или теги года, удаляются перед поиском).
2. Точное совпадение с полным именем исполнителя и оригинальным названием (если очистка его изменила).
3. Точное совпадение только с именем первого исполнителя и очищенным названием (для строк с несколькими исполнителями, например «Исполнитель A & Исполнитель B»).
4. Нечёткий поиск с именем первого исполнителя и очищенным названием.
5. Нечёткий поиск с исходной строкой трека в качестве крайнего средства.

Если доступны простые тексты, они отображаются как есть. Если доступны только синхронизированные по времени LRC-тексты, временны́е метки удаляются и отображается простой текст. Инструментальные треки сообщаются как не найденные.


## Воспроизведение

Дополнение выбирает движок воспроизведения, следуя следующему приоритету:

1. **BASS** — основной движок по умолчанию. Не требует отдельной установки; встроен в дополнение. BASS отправляет аудио напрямую в аудиостек Windows и отображается в микшере громкости Windows как независимый источник звука с именем «pythonw.exe», отдельно от NVDA. Это означает, что звук FreeRadio проходит по совершенно отдельному каналу, не смешиваясь с речью NVDA: радио не прерывается, не смешивается и не зависит от собственных аудионастроек NVDA, когда NVDA говорит. Пользователь может регулировать громкость радио независимо от NVDA в микшере громкости Windows. Поддерживает HTTP, HTTPS и большинство встроенных форматов потоков. Аудиозеркало доступно только с этим движком.
2. **VLC** — вступает в действие, если BASS не работает. Поиск выполняется в стандартных местах установки, папках профиля пользователя и системной переменной PATH.
3. **PotPlayer** — используется, если VLC не найден. Поиск выполняется в стандартных местах установки.
4. **Windows Media Player** — используется в крайнем случае; требует, чтобы компонент WMP был установлен в системе.

## Проверка обновлений

FreeRadio автоматически проверяет наличие новых версий через GitHub.

**Автоматическая проверка:** Выполняется в фоновом режиме через 15 секунд после запуска NVDA. Если найдена новая версия, вы получаете уведомление; если версия актуальна, сообщение не показывается.

**Ручная проверка:** Может быть запущена по запросу из меню NVDA → Сервис → FreeRadio → **Проверить обновления…**. При запуске таким способом результат объявляется, даже если версия актуальна.

**Когда найдено обновление:** Открывается диалоговое окно с указанием номера версии и вашей установленной версии.

- Если на странице релиза GitHub доступен непосредственно загружаемый файл `.nvda-addon`, отображается кнопка **Загрузить и установить**. После подтверждения файл загружается в фоновом режиме, NVDA объявляет о начале загрузки, и автоматически открывается окно установки NVDA.
- Если прямая ссылка для загрузки недоступна, отображается кнопка **Открыть страницу**, и страница релиза GitHub открывается в браузере по умолчанию.

**Чтобы отключить автоматические проверки:** Выключите опцию **Проверять обновления автоматически** в меню NVDA → Параметры → Настройки → FreeRadio.
